## âš™ï¸ Installation & Setup

### Prerequisites

- Node.js (v16 or later recommended)
- MongoDB (local or cloud)
- Git

### Clone the Repository

```bash
git clone https://github.com/ferdoushasanpr/LuminaBazaar-React-Based-Ecommerce-Application.git
cd LuminaBazaar-React-Based-Ecommerce-Application
npm install
npm run dev
```

### Backend Setup

```bash
git clone https://github.com/ferdoushasanpr/luminabazzar-restapi.git
cd luminabazzar-restapi
npm install
node index
```
